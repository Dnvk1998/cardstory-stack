# Scroll Story Stack

A GSAP-free, cinematic scroll storytelling system inspired by Apple product pages.

## Features
- Scroll-driven card storytelling
- Section pinning
- Progress bar + timeline
- Infinite cards
- Mobile friendly
- Zero dependencies

## Use cases
- Landing pages
- SaaS product stories
- Portfolios
- Marketing sites

## How to use
1. Open `source/index.html`
2. Replace card content
3. Adjust story length via body height
4. Deploy anywhere

## Requirements
- Modern browser
- No build tools
